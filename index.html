☁ 雲端同步設定

連接 Supabase 後，你的資料會自動在所有裝置上同步。
設定步驟
1. 去 supabase.com 免費註冊 → 建立新 Project
2. 進入 Project → Settings → API
3. 複製 Project URL 和 anon public key
4. 進入 SQL Editor，執行以下指令建表：
create table alfred_store (
  user_id text primary key,
  data jsonb not null,
  updated_at timestamptz default now()
);
alter table alfred_store enable row level security;
create policy "open" on alfred_store
  for all using (true) with check (true);
